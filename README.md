## Téléchargement
Rendez-vous dans l'onglet release et téléchargez l'exécutable `NetCalc.exe` (https://github.com/kooorvo/NetCalc-IP/releases)

### Instructions

#### Windows
  1. Téléchargez le fichier `.exe`.
  2. Exécutez-le (aucune installation requise).

#### Linux
  1. Téléchargez le fichier `Source Code`
  2. DÉcompressez l'archive
  3. Ouvrez le fichier `root.py` dans un éditeur de code (Syper, Visual Studio Code, autres...)
  4. Exécutez le programme
